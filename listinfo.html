About

The Mailman Beautification Project is a mailing list for showcasing, discussing, and building more useable and beautiful mailing list information pages for the Mailman email list package. Join us!

If you want to help improve this page (and others like it), its source can be found on Github. Also, come hang out in Mailman's IRC channel (#mailman on irc.freenode.net)!
